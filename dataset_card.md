Dataset: Shinchan Character Face Dataset

Source:
Frames extracted from publicly available Shinchan videos.

Method:
- Videos downloaded with yt-dlp (720p)
- Frames extracted at 2 FPS using ffmpeg
- Anime face detection using YOLO anime-face model
- Faces cropped and resized to 256x256
- Duplicate removal using perceptual hashing
